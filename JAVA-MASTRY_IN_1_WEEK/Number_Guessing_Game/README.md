# 🎯 Project 2: Number Guessing Game

## 📚 Concepts Used

| Concept | Description |
|---------|-------------|
| **Random** | `Random` class to generate random numbers |
| **Loops** | `while` loop to keep guessing until correct |
| **Conditionals** | `if-else` to check guess vs target number |
| **User Input** | `Scanner` class for keyboard input |
| **Methods** | Organizing code into reusable functions |
| **Variables** | Storing game state (target number, attempts, etc.) |
| **Math Operations** | Using `Math.random()` as alternative to Random class |

## ✨ Features

- 🎲 Computer generates a random number (1-100)
- 🔢 Player guesses the number
- 📈 Hints: "Too High!" or "Too Low!"
- ⏱️ Tracks number of attempts
- 🔄 Play again option
- 🏆 Win celebration

---

## 💻 How to Run

```bash
javac GuessingGame.java
java GuessingGame
```

---

## 📖 Learning Notes

This project teaches you:
1. **Random class** - Generate random numbers for games
2. **While loop** - Repeat until condition is met (correct guess)
3. **If-else** - Make decisions based on conditions
4. **Scanner** - Read user guesses
5. **Methods** - Organize game logic into functions
6. **Variables** - Track game state (target, attempts, etc.)
