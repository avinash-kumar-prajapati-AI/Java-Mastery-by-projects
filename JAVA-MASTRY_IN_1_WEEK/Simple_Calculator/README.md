# 🧮 Project 1: Simple Calculator

## 📚 Concepts Used

| Concept | Description |
|---------|-------------|
| **Class** | Blueprint for creating objects — defines `Calculator` as a template |
| **Methods** | Functions inside a class that perform operations (`add`, `subtract`, etc.) |
| **Operators** | Mathematical operators: `+`, `-`, `*`, `/`, `%` |
| **User Input** | Using `Scanner` class to read user input from console |
| **Basic I/O** | Input via keyboard, output to console using `System.out.println()` |
| **Control Flow** | Using `switch` statement to handle menu choices |
| **Loops** | Using `do-while` loop to keep the calculator running until user exits |

## 📖 Learning Notes

This project introduces you to the **basic structure of a Java program**:
1. Every Java program starts with a `class`
2. The `main` method is the entry point (`public static void main(String[] args)`)
3. `Scanner` is used to read input from the user
4. Methods organize code into reusable blocks
5. `switch` handles multiple menu options cleanly
