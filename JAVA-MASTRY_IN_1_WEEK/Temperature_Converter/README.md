## 📖 Learning Notes

This is your first **true OOP project**! You'll learn:

1. **Encapsulation** - Hide internal data with `private`, expose with `public`
2. **Constructor** - Initialize object state when created
3. **Instance Variables** - Each object has its own temperature value
4. **Methods** - Perform conversions (Celsius → Fahrenheit, etc.)
5. **Getters** - Access data through methods, not directly

### Why Encapsulation?
- 🔒 Protects data from invalid values
- 📦 Hides implementation details
- 🔧 Easier to modify code later
- ✅ Makes code more maintainable
