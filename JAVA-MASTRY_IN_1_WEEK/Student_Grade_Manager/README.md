# 🎓 Project 4: Student Grade Manager

## 📚 Concepts Used

| Concept | Description |
|---------|-------------|
| **ArrayList** | Dynamic list for storing students and grades |
| **Arrays** | Fixed-size storage for simple grade values |
| **Loops** | Repeat actions while displaying or processing data |
| **Methods** | Organize logic into reusable functions |
| **Objects** | Represent each student as an object |
| **Encapsulation** | Keep student data safe inside the class |
| **Collections** | Use Java collections to manage multiple items |

```
Student_Grade_Manager/
```

## ✨ Features

- 🧑‍🎓 Add students
- 📝 Store grades for each student
- 📊 Calculate average grade
- 🔎 Display all students and their results
- ✅ Find highest and lowest grades

## 💻 How to Run

```bash
javac StudentGradeManager.java
java StudentGradeManager
```

---

## 📖 Learning Notes

This project helps you practice:

1. **ArrayList** for growing collections of students
2. **Arrays** for grade values of a single student
3. **Methods** for adding, listing, and calculating results
4. **Objects** to model a student clearly
5. **Encapsulation** by keeping data private and safe
